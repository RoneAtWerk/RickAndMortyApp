# RickAndMortyApp

RickMortyApp is a SwiftUI app built with the MVVM pattern.  
It uses the Rick and Morty API to display characters in a list and allows the user to tap a character to view more details.

## Features
- Fetches character data from a public API
- Displays characters in a SwiftUI List
- Navigates to a detail view
- Uses MVVM architecture

## Technologies Used
- Swift
- SwiftUI
- MVVM
- URLSession
- Rick and Morty API

## API Used
https://rickandmortyapi.com/

## Screens
- Character List Screen
- Character Detail Screen
